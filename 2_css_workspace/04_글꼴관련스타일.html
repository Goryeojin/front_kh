font-family : 글꼴을 지정할 때 사용

        [ 표현법 ]
        선택자 {
            font-family : 바꾸고자하는 글꼴명;
        }

font-size: 글꼴의 크기를 변경할 때 사용

        [ 표현법 ]
        선택자 {
            font-size : 크기에 해당되는 숫자(단위: px/em/%);
        }
  • px 테스트
  • em 테스트
  • % 테스트

font-weight : 글꼴의 굵기를 표현할 때 사용

        [ 표현법 ]
        선택자{
            font-weight : normal / bold / border / lighter / 100 / 900;
        }
  • 원래 굵기
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 굵게
  • 원래 굵기보다 더 가늘게

font-style :텍스트문구를 기울이고자 할 때 사용

        [ 표현법 ]
        선택자 {
            font-style : normal / italic / oblique;
        }

        => italic : 기울임 글꼴, oblique : 원래 글자를 기울임
  • italic : 기울임 글꼴 테스트
  • oblique : 원래 글자를 기울임
