목록 관련 스타일 (list~)

list-style-type : 불릿 기호를 바꿀 때 사용 

        [ 표현법 ]
        * 순서 없는 목록 (ul)
        선택자 {
            list-style-type : disc/circle/square/none;
        }

        * 순서 있는 목록 (ol)
        선택자 {
            list-style-type : decimal => 아라비아 숫자(십진수)
            list-style-type : lower-alpha/upper-alpha => 알파벳
            list-style-type : lower-roman/upper-roman => 로마숫자
        }

순서 없는 목록

  • HTML
  • CSS
  • JavaScript
  • jQuery

순서 있는 목록

  1. HTML
  2. CSS
  3. JavaScript
  4. jQeury

list-style-position : 불릿기호의 위치를 조정할 때 사용

list-style-image : 불릿기호로 이미지를 적용할 때 사용

        [ 표현법 ]
        선택자 {
            list-style-position : inside/outside;
            list-style-image : url(적용시키고자 하는 이미지 경로);
        }
