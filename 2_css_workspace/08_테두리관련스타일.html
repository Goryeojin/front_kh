테두리 관련 스타일 (border~)

        [ 표현법 ]
        선택자 {

            * 테두리 스타일
            border-위치-style : none/dotted/dashed/solid/double/groove/ridge/inset/outset;

            * 테두리 두께
            border-위치-width : 테두리두께px;

            * 테두리 색상
            border-위치-color : 테두리색상;

            * 테두리 모서리를 동그랗게
            border-top/bottom-left/right-radius: 숫자px; => 위치 생략가능 (사방이 동그람)

            * 박스에 그림자
            box-shadow : 가로(x) 세로(y) 흐림정도 번짐정도 색상;
        }
