함수(function)

* 선언적 함수

        [표현법]
        function 함수명(매개변수, 매개변수, ...){
            
            해당 함수를 호출할 경우 실행할 코드;

            return 결과값;
        }

        - 매개변수가 없는 경우 생략 가능
        - 반환할 결과값이 없는 경우 return 결과값;도 생략 가능

* 익명 함수

        [표현법]
        변수 = function(){

            소스코드 어쩌고 저쩌고;

        }
        - 매개변수 생략 가능
        => 특정 변수나 속성에 대입하는 함수가 필요할 때 주로 사용됨
        (주로 이벤트 핸들러 작성시 사용)
